Apps, Tools & Devices

AWI

iPhone Android

Smart, AI-integrated assistance for daily living — designed first for low-vision and blind users, helpful for everyone. AWI combines spatial awareness cues, context-sensitive prompts, and clear auditory feedback to make moving through the world simpler and safer. Think turn-by-turn awareness, gentle proximity alerts, and hands-free interactions that reduce cognitive load rather than add to it.

Bright Scan

macOS Windows

A lightweight, efficient site scanner that flags technical issues and notifies you proactively. Optional add-ons include GDPR-aware checks, accessibility-friendly recommendations, uptime pings, and performance snapshots — so you get fast, practical guidance instead of noisy reports.

BCL AI Smart Speaker

Product

A friendly smart speaker with selectable personalities and the core features you’d expect — timers, music, answers — plus a focus on clarity and inclusive interaction. Designed to be helpful out of the box without feeling clinical.

Cursor Vox

macOS Windows

A compact voice-over companion for low-vision and blind users to navigate desktops more comfortably. Cursor Vox focuses on clear speech, quick context hints, and smart shortcuts so routine tasks take fewer steps.

DevM8

macOS Windows

A beginner-friendly coding companion that helps new developers learn by doing. DevM8 offers guided, click-to-place building blocks, context tips, and previews — so you can assemble working snippets and understand the “why” behind them.

The Barkers

macOS Windows

Meet the Barkers — an AI-driven, slice-of-life family you can drop in on like a friendly neighbour. Light, conversational, and curious: a playful exploration of ambient AI characters.
